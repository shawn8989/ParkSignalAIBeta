# New Features Summary

## Alarm Testing
- Added capability to test alarms directly within the application.
- Allows users to verify alarm configurations without triggering real alerts.
- Usage: Navigate to the Alarm section and select "Test Alarm" to simulate and review alarm behavior.

## Scans Management
- Enhanced scan scheduling and management features.
- Users can create, edit, and delete scheduled scans.
- Usage: Access the Scans tab to manage scan tasks; enables better control over scanning routines.

## History Management
- Improved history tracking and management.
- Users can view detailed logs of past activities and clear history records.
- Usage: Go to the History section to review logs or clear history as needed.

## Dashboard Declutter with Links
- Dashboard interface has been simplified for better user experience.
- Key sections are now accessible via quick links, reducing clutter.
- Usage: Use the provided links on the dashboard to quickly navigate to important areas.

## Settings Enhancements
- Settings menu reorganized for easier access and configuration.
- New options added to customize application behavior.
- Usage: Open Settings to adjust preferences and configure new options according to your needs.

---

This summary highlights the latest improvements aimed at enhancing usability and functionality across the application.
